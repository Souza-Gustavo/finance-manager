# Finance Manager – Backend (Spring Boot)

Projeto backend para gestão financeira pessoal, desenvolvido em Java com Spring Boot.

## Funcionalidades implementadas (MVP)
- Cadastro e autenticação de usuários (JWT)
- Criação de parcelas (installments)
- Listagem de parcelas por usuário autenticado
- Segurança com Spring Security + JWT
- Persistência com JPA / Hibernate
- Banco H2 (ambiente de desenvolvimento)

## Tecnologias
- Java
- Spring Boot
- Spring Security
- JWT
- JPA / Hibernate
- H2 Database
- Maven
