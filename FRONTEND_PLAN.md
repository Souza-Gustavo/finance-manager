# Frontend – Finance Manager

## Objetivo
Criar um frontend web para consumo da API Finance Manager, com autenticação JWT e foco em simplicidade e usabilidade.

## Telas iniciais (MVP)
1. Login
2. Cadastro
3. Dashboard
4. Parcelas (listagem + criação)

## Fluxo de navegação
- Usuário não autenticado → Login
- Login bem-sucedido → Dashboard
- Dashboard → Parcelas

## Componentes principais
- Input
- Button
- Card
- Navbar

## Comunicação com backend
- API REST
- JWT armazenado no localStorage
- Axios com interceptor

## Tecnologias planejadas
- React
- Vite
- Axios
- CSS puro (inicialmente)
